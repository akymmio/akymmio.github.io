符号

  • ( ) 可以使用$() 将命令赋值给一个变量
  • 最基本的条件测试语法,用于文件测试和比较字符串
  • [[ ]] 比 [ 和 ] 更强大，支持正则表达式和更复杂的字符串操作
  • (( )) 算术运算 的语法，用于数值计算和比较。
  • 命令执行

  • 注释

[ 和 ] 或 [[]] 进行条件测试时，它们周围必须有空格

关于单双引号的问题：

  • 双引号能够识别变量，双引号能够实现转义（类似于“*”）

  • 单引号是不能识别变量，只会原样输出，单引号是不能转义的

  • 文件测试：test 或[ ], [ -f file ] 检查文件是否存在且为普通文件。

  • 字符串测试：[ “string1” = “string2” ] 检查两个字符串是否相等。

  • 算术比较：(( num1 > num2 )) 检查 num1 是否大于 num2。

关系运算符

关系运算符只支持数字，不支持字符串，除非字符串的值是数字

字符串比较

  • = 或 ==：检查字符串是否相等。
  • !=：检查字符串是否不相等。
  • z：检查字符串是否为空。
  • n：检查字符串是否非空。
