变量

使用时需要加上$，赋值和$(( ))中不用加

  • 系统变量

  • 环境变量

  • 用户变量

变量删除： unset 变量名

环境变量： export

只读变量： readonly

数组

符号

  • ( ) 可以使用$() 将命令赋值给一个变量
  • 最基本的条件测试语法,用于文件测试和比较字符串
  • [[ ]] 比 [ 和 ] 更强大，支持正则表达式和更复杂的字符串操作
  • (( )) 算术运算 的语法，用于数值计算和比较。
  • 命令执行

  • 注释

[ 和 ] 或 [[]] 进行条件测试时，它们周围必须有空格

关于单双引号的问题：

  • 双引号能够识别变量，双引号能够实现转义（类似于“*”）

  • 单引号是不能识别变量，只会原样输出，单引号是不能转义的

  • 文件测试：test 或[ ], [ -f file ] 检查文件是否存在且为普通文件。

  • 字符串测试：[ “string1” = “string2” ] 检查两个字符串是否相等。

  • 算术比较：(( num1 > num2 )) 检查 num1 是否大于 num2。

运算符

逻辑运算符

  • ! 非
  • -a 与
  • -o 或

用于[ ]

&& || 要在 [[ ]] (( )) 中使用

变量操作${}

多命令执行

  • ； (命令1 ；命令2)
  • &&
  • ||

关系运算符

关系运算符只支持数字，不支持字符串，除非字符串的值是数字

字符串比较

  • = 或 ==：检查字符串是否相等。
  • !=：检查字符串是否不相等。
  • z：检查字符串是否为空。
  • n：检查字符串是否非空。

算数运算

$(( )) ： j=$((i + j))

与 $(( )) 类似，(( )) 语法也可以用于算术运算

expr : val=expr 2 + 2 expr $a \* $b 运算符两边需要空格、一些运算符需要转义

函数

  • exit

I/O

  • 输出重定向：>
  • 追加 ： >>
  • &> : 正确和错误都输出

不输出：command > dev/null

cut

字符截取: cut [选项] 文件名

选项:
-f 列号: 提取第几列

-d 分隔符: 按照指定分隔符分割列
-n 取消分割多字节字符
-c 字符范围

awk

可以多个条件{动作}，多个动作之间可以用 ; 分开
