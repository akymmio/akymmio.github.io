Linux

Created: July 30, 2024 8:55 AM
Tags: linux

首次安装完系统之后需要设置

sudo passwd 设置root密码

2.

apt install openssh-server

service ssh start 开启ssh

systemctl status sshd 检查是否开启

sudo vim /etc/ssh/sshd_config

开启root用户ssh连接: #PermitRootLogin prohibit-password→PermitRootLogin yes

systemctl restart sshd 重启ssh服务

sudo apt update 更新软件
sudo apt upgrade

换源 /etc/apt/sources.list

拓展

ll 展示的文件名后面带一个*说明具有可执行权限

ssh免密连接

  1. ssh-keygen -t rsa 生成密钥对 （C:\Users\用户.ssh）
  2. 将公钥写入目标服务器中（~/.ssh/authorized_keys ）

拉取文件

scp username@userhost:/path /destination_path

（确保有权限操作目标文件）
