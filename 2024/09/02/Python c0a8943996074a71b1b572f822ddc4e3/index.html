  • int

字符串

  • Python中的字符串用单引号 ‘ 或双引号 “ 括起来
  • 索引值以 0 为开始值，-1 为从末尾开始。
  • 字符串不能被改变
  • +连接 *重复输出字符串
  • [ : ] 截取字符串中的一部分

方法：

  • find()
  • index
  • lower() upper()
  • replace(old, new [, max])
  • split(str=””, num=string.count(str)) 分隔符，次数
  • startswith(substr, beg=0,end=len(string))

List列表

列表[]可以完成大多数集合类的数据结构实现，一个列表可以存储各种类型的数据

  • 列表中的元素可以改变
  • 可以使用+拼接
  • 可以被索引和切片

方法：

  • append()
  • count()
  • index()
  • insert()
  • reverse()
  • remove()
  • pop([index=-1])删除最后一个列表值。
  • sort( key=None, reverse=False) ,key是函数，排序规则
  • list() 元组转列表
  • clear()
  • copy()
  • extend()追加
  • del 删除，del list[2]

元组tuple

(,)元组不可变,不可删，但是可以组合(元组的不可变指的是元组所指向的内存中的内容不可变)

任意无符号的对象，以逗号隔开，默认为元组

tup1 = (50,)元组中只包含一个元素时，需要在元素后面添加逗号,否则括号会被当作运算符使用

元素值是不允许删除的，但我们可以使用del语句来删除整个元组

集合set

一种无序、可变的数据类型，用于存储唯一的元素

集合使用大括号 { } 表示，元素之间用逗号, 分隔。

也可以使用 set() 函数创建集合

注意：创建一个空集合必须用 set() 而不是 { }，因为 { } 是用来创建一个空字典。

set可以进行集合运算

  • remove()
  • discard() 删除,不存在不报错
  • pop() 随机删除
  • update(),add()

字典dict

{ : }, 键的类型为不可变,值没有限制

  • str()打印字符串表示
  • get()
  • keys(),values(),items()
  • pop()
  • dict.setdefault(key, default=None)
  • dict.update(dict2) 更新（没有的添加，有的更新
  • dict.fromkeys(seq[, val]) 创建一个新字典
  • del 删除元素
  • 通过类似数组操作方法添加元素

遍历：默认遍历键

bytes

在 Python3 中，bytes 类型表示的是不可变的二进制序列

使用

多变量赋值： a,b=1,2

除法：/返回浮点数 //返回整数向下取整

** 指数

* 复制

推导式

推导式是一种独特的数据处理方式，可以从一个数据序列构建另一个新的数据序列的结构体。

推导式是一种强大且简洁的语法，适用于生成列表、字典、集合和生成器

每个列表推导式都在 for 之后跟一个表达式，然后有零到多个 for 或 if 子句

装饰器

Python 装饰允许在不修改原有函数代码的基础上，动态地增加或修改函数的功能，装饰器本质上是一个接收函数作为输入并返回一个新的包装过后的函数的对象。

类型转换

  • str(x) 转字符串类型
  • chr(x) 将一个整数转换为一个字符
  • ord(x) 将一个字符转换为它的整数值
  • hex(x) 将一个整数转换为一个十六进制字符串
  • oct(x) 将一个整数转换为一个八进制字符串

遍历方式

range() 函数。它会生成数列,还可以使用 range() 函数来创建一个列表:list(range(5))

range(start,stop) 左闭右开

range(start,stop,step) 起始值 结束值 步长

逻辑运算符 and or not

成员运算符

  • in 如果在指定的序列中找到值返回 True，否则返回 False。
  • not in 如果在指定的序列中没有找到值返回 True，否则返回 False。

身份运算符

  • is , is not 判断两个标识符是不是引用自不同对象

is 与 == 区别：is 用于判断两个变量引用对象是否为同一个， == 用于判断引用变量的值是否相等。

__private_attrs：两个下划线开头，声明该属性为私有，不能在类的外部被使用或直接访问。在类内部的方法中使用时 self.__private_attrs。

__private_method：私有方法

  • 单个前置下划线 _ 表示这是一个内部使用的方法，不建议外部直接访问。
  • 双前置下划线 __ 表示这是一个真正私有的方法，其名称会被进行名称改编。

调用表达式:根据条件动态选择并调用不同的类或函数

rv = (self.loggerClass or _loggerClass)(name)

  • (self.loggerClass or _loggerClass)(name) 是一个函数调用表达式。
  • 选择出来的类（self.loggerClass 或 _loggerClass）被当作 callable 对象调用，并传递参数 name。
