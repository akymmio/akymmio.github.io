vue

Vue是一款用于构建用户界面的 JavaScript 框架。它基于标准 HTML、CSS 和 JavaScript 构建，并提供了一套声明式的、组件化的编程模型，帮助你高效地开发用户界面。无论是简单还是复杂的界面，Vue 都可以胜任。

Vue.js 提供了MVVM数据绑定和一个可组合的组件系统

概念

MVVM,模型-视图-视图模（Model-View-ViewModel）的缩写，是一种设计模式，经常用于创建用户界面

__DOM__：文档对象模型，是一个网络文档的编程接口。它代表页面，以便程序可以改变文档的结构、风格和内容。DOM 将文档表示为节点和对象；这样，编程语言就可以与页面交互。

创建vue项目

  • 创建cmd文件，运行vue ui
  • 打开Router功能，选择好vue版本，创建项目
  • 运行项目：npm run serve

Vue程序结构

  • node_modules 项目依赖包

    • index 模板文件
  • src 项目源代码

  • package.json 项目配置文件（依赖、版本号等）
  • vue.config.js vuep配置信息

只要有 .json 文件，通过命令 npm install 就可以下载依赖，

new Vue({}) 和 createApp({}) 是在 Vue.js 框架中创建新的 Vue 应用程序的两种方法。

new Vue({})：这是 Vue 2.x 版本中创建新 Vue 实例的方式。在这个版本中，Vue 实例是全局的，并且可以在任何地方访问和使用。

createApp({})：这是 Vue 3.x 版本中创建新 Vue 应用程序的方式。这个版本的 Vue 应用程序更加模块化，并且每个应用程序都有一个独立的 Vue 实例。

vue 操作

常用指令

指令:带有v-前缀的特殊标签属性

  • v-bind 为HTML标签绑定属性值
    • v-bind:属性名=”表达式”
    • :属性名=”表达式”
  • v-model 在表单元素上创建双向数据绑定
  • v-on (简写@) 为HTML标签绑定事件
    • v-on: 事件=”内联语句”
    • v-on: 事件=”函数（args）”
  • 条件性渲染
    • v-if 创建或移除
    • v-else-if=”表达式”
    • v-else
    • v-show 显示或隐藏（适用于切换频繁的）
  • v-for 列表渲染，遍历容器元素或对象属性
    • v-for=”(item,index) in 数组|对象”
  • v-html 解析html标签

其他

  • props:[] : 自定义属性，从外部传入组件数据
  • this: vue对象，可以访问数据
  • => :箭头函数，作用域继承于父级（不会让回调函数中this找不到）

指令修饰符

注意事项

函数中直接使用变量会被认为是使用了全局变量，然而并没有这个全局变量，会报错，需要添加this
（即app.变量）
this被规定指向当前实例

v-for中的key：会给元素添加唯一标识，便于vue进行列表项的正确排序复用（推荐使用id）

方法名不能有特殊字符，有的话要用单引号括起来

vue组件

  • 页面组件（放views文件夹）
  • 复用组件（放components文件夹）

组件化：一个页面可以拆分成一个个组件，每一个组件有自己独立的结构、样式和行为，组件可以复用，从而提高开发效率和代码可维护性。

根组件(App): 整个应用的入口组件，用于创建应用实例，并挂载到DOM中。

  • <template> 模板部分，生成HTML代码
  • <script> 控制数据的来源和行为，js
  • <style> css样式
