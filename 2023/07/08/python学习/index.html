数据类型

  • Number
  • String
  • Tuple(元组)
  • bool
  • List
  • Set(集合)
  • Dictionary(字典)
    （前三个为可变数据，后三个为不可变数据）

Number

int、float、bool、complex(复数)
type(a)、isinstance(a,int)查询变量所指的对象类型
del var 删除单个或多个对象

  • 多变量赋值 a,b=1,2
  • 除法：/返回浮点数 //返回整数

String

  • Python中的字符串用单引号 ‘ 或双引号 “ 括起来
  • 索引值以 0 为开始值，-1 为从末尾的开始位置。
  • 字符串不能被改变
  • +连接 *复制

List列表

列表可以完成大多数集合类的数据结构实现

变量[头下标：尾下标]
list = [ 'abcd', 786 , 2.23, 'runoob', 70.2 ]
  • 列表中的元素可以改变
  • 可以使用+拼接
  • 可以被索引和切片
