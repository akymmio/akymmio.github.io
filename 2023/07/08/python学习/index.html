字符串

  • Python中的字符串用单引号 ‘ 或双引号 “ 括起来

  • 字符串不能被改变
  • +连接 *重复输出字符串
  • [ : ] 截取字符串中的一部分
  • 三引号允许一个复杂字符串跨多行
