web前端

javascript

JavaScript 是一个脚本语言。

它是一个轻量级，但功能强大的编程语言。

弱语言类型

引用

  • 内部
    • <script> </script> 之间
    • 可在html文档中任意位置
    • 一般把脚本放在<body>元素底部
  • 外部
    • 外部js文件中，不含<script>标签
    • <script>不能自闭合

变量

var，全局变量，可重复定义
let，局部变量，不可重复定义，只在 let 命令所在的代码块 {} 内有效。

  • 未使用值来声明的变量，其值实际上是 undefined
  • 重新声明 JavaScript 变量，该变量的值不会丢失
  • 变量的数据类型可以使用 typeof 操作符来查看
  • 可以使用关键词 “new” 来声明其类型：
    • var carname=new String;

数据类型

基本类型

  • number
  • string
  • boolean
  • null
  • undefine

引用类型
对象(Object)

数组(Array)

函数(Function)

运算符

  • 算数运算符
  • 赋值运算符
  • 比较运算符
  • 逻辑运算符
  • 三元运算符

=== 比较值和类型

类型转换

  • 其他类型转换为number：数字字符转字面值，非数字字符转为NaN
  • 其他类型转换为boolean：0和NaN以为外的其他数字转为true，其他字符串转化为true，空字符串为false
  • null、undefine转化为false

函数

    function functionName(参数1，参数2...){
            //code
    }

js对象

自定义对象：

    var object{
            属性：属性值,
            函数名：function(){}
    };

  • Array:长度可变，类型可变，默认初始化为undefine，

    • length:求长度
    • push:添加元素
    • splice：删除元素，splice(2,2)删除下标为2开始的往后2个元素
    • forEach:遍历数组中有值的函数

  • String

    • length:字符串长度
    • indexOf():检索字符串
    • trim():去除字符两边的空格
    • substring():提取字符串

  • json(JavaScript Object Notation)

    • json是通过js对象标记法书写的文本

    • 作为数据载体，在网络中进行数据传输

      var 变量名='{"key1":value1,"key2":value2}'

    • json字符串转为js对象：JSON.parse();

    • js对象转化为json字符串:JSON.stringify();

  • bom(browser object model)允许js与浏览器对话，将浏览器各个组成部分封装成对象

    • window
    • location.href

  • DOM(文档对象模型，将标记语言的各个组成部分封装为对应的对象)
    javascript通过DOM对HTML进行操作

    • Document
    • Element
    • Attribute
    • Text
    • Comment

获取Element元素对象
document.getElementById();id属性
getElementsByTagName();标签名
getElementsByName();name属性
getElementsByClassName();class属性

常见的HTML事件

下面是一些常见的HTML事件的列表:

事件

  • onchange HTML 元素改变
  • onclick 用户点击 HTML 元素
  • onmouseover 鼠标指针移动到指定的元素上时发生
  • onmouseout 用户从一个 HTML 元素上移开鼠标时发生
  • onkeydown 用户按下键盘按键
  • onload 浏览器已完成页面的加载

vue

基于MVVM模型地前端js框架，免除js中的DOM操作,简化书写

常用指令

  • v-bind 为HTML标签绑定属性值
  • v-model 在表单元素上创建双向数据绑定
  • v-on (简写@) 为HTML标签绑定事件
  • 条件性渲染
    • v-if 创建或移除
    • v-else-if
    • v-else
    • v-show 显示或隐藏（适用于切换频繁的）
  • v-for 列表渲染，遍历容器元素或对象属性

生命周期

  • mounted()：vue挂载完成

Ajax

异步的js和XML

  • 数据交换：通过Ajax向服务器发送请求，并获取服务器响应地数据
  • 异步交互：不重新加载页面的情况下，与服务器交换数据并更新部分网页，如：联想搜索和用户名校验

Ajax-Axios

对原生Ajax封装，简化书写，快速开发

  • 引入Axios的js文件
  • 使用Axios发送请求，获取响应结果

请求方式别名

  • axios.get(url[,config]);
  • axios.post(url[,data[,config]]);
  • axios.delete();
  • axios.put();

Promise

Promise:是一个对象，可以获取异步操作的消息， 解决异步回调地狱问题，

Promise的三种状态
