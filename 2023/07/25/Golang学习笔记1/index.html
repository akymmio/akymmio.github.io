Go语言

特点

  1. 高性能、高并发
  2. 语法简单、学习曲线平缓
  3. 丰富的标准库
  4. 完善的工具链
  5. 静态链接
  6. 快速编译
  7. 跨平台
  8. 垃圾回收

配置golang开发环境

基础语法

golang是强类型语言

变量

变量声明：声明语句定义了程序的各种实体对象以及部分或全部的属性。Go语言主要有四种类型的声明语句：var、const、type和func，分别对应变量、常量、类型和函数实体对象的声明。

    var 变量名字 类型 = 表达式

  • var 变量名=值

  • 变量名:=值

      var a="initial"

  var e float64
  f:=float32(e)

if-else

  • if后面没有括号，后面必须接{ }

循环

Go语言只有for循环这一种循环语句。for循环有多种形式，不需括号包围。

    for j:=1;j<10;j++{
        
    }

    for{
        //无限循环
    }

for循环的另一种形式，在某种数据类型的区间（range）上遍历，如字符串或切片。

switch

默认不需要break跳出case，可以实现更好地if-else

数组

数组是一个由固定长度的特定类型元素组成的序列，一个数组可以由零个或多个元素组成

如果一个数组的元素类型是可以相互比较的，那么数组类型也是可以相互比较的，这时候我们可以直接通过==比较运算符来比较两个数组

    var a[5]int
    var b [3]int = [3]int{1, 2, 3}
    q := [...]int{1, 2, 3}  //长度是根据初始化值的个数来计算

切片

Slice（切片）代表变长的序列，序列中每个元素都有相同的类型。

和数组不同的是，slice之间不能比较，因此我们不能使用==操作符来判断两个slice是否含有全部相等元素。唯一合法的比较操作是和nil比较

  s:=make([]string,3)//创建数组
  s=append(s,"d");   //添加元素

map

    m:make(map[string]int) //创建map 
    r,ok:=m["unknow"]      //根据key查找，返回Boolean
    delete(m,"one")        //删除key="one"的元素

range

遍历

    遍历数组
    for i,num:=range nums{
            //i是下标，num是对应的元素
    }

    遍历map
    for k,v:=range m{
            //k是key v是value
    }
    for k:=range m{
            //k是key
    }

函数

变量值后置，可以有多个返回值（第一个是value，第二个是Boolean） 

    func add(a int,b int) int{
            return a+b
    }

    func exist(m map[string]string,k string) (v string, ok bool){
            //前一个括号是参数，后一个括号是返回值
            v,ok=m[k]
            return v,ok
    }
    //m[k] 的方式从 map 中获取键 k 对应的值，并将结果赋值给 v 和 ok。如果键 k 存在于 map 中，则 ok 的值为 true，否则为 false。

指针

一个指针的值是另一个变量的地址。一个指针对应变量在内存中的存储位置。
go中用于对传入的参数进行修改

    *p
    new(T)

结构体

    type user struct{
            name     string
            password string
    }

    初始化方式：
    a:=user{name:"wang" password:"1234"}
    b:=user{"wang","1234"}
    c:=user{name:"wang"}

    var d=user;
    d.name="wang"
    d.password=1234

结构体方法

    a:=usr{"wang","1234"}
    a.resetPassword("2048");

    func(u *user) resetPassword(password string){
            //传入指针
            u.password=password
    }

错误处理

字符串操作

  • Contains() 是否含有字符串
  • count() 字符串计数
  • Index() 字符串位置
  • Join() 连接字符串
  • Repeat() 重复字符串
  • len() 字符串长度

字符串格式化

Println() 打印换行
Printf(“%v”,a) %v打印任意类型变量
Printf(“%+v”,a) 打印字段名字和值（a是结构体）
Printf(“%#v”,a) 打印结构体构造名称、字段和值

json处理

    type userInfo struct{
            //字段开头大写
            Name string
            Age  int 'json:"age"' //将字段名改为age
            Hobby []string
    }

    a:=userInfo{Name:"wang",Age:18,Hobby:[]string{"Golang","TypeScript"}}

    buf,err:=json.Marshal(a)//序列化，转化为字符串数组
    fmt.Println(string(buf))

    var b userInfo
    err=json.Unmarshal(buf,&b)//反序列化

时间处理

now:=time.Now() 获取当前时间
Year() Month() Day() Hour() Minute() 获取时间点信息
t2.Sub(t1) 两个时间段之差
now.Unix() 获取时间戳

数字解析（strconv）

ParseInt(“111”,10,64) 将字符串 “111” 解析为十进制的 64 位有符号整数。
Atoi(“123”) 十进制字符串转化为数字
Atoi(“AAA”)

进程信息

os.Args 获取命令行参数
os.Getenv()/setenv() 获取和设置环境变量
exec.Command 创建一个表示外部命令的 *exec.Cmd 结构体
