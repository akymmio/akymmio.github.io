git区

  • 工作区
  • 暂存区
  • 本地库
  • 远程库

git撤销提交

add 未commit

commit 未push

push

mixed: 取消暂存区，保留工作区
soft:不会改变暂存区或工作区的文件
hard:彻底重置暂存区和工作区

命令

撤销

git restore 取消工作区操作
git reset 重置，版本回退
git revert 反向操作，版本回退

分支操作

git branch 创建分支
git branch -d 删除分支
git checkout 切换分支
git meger 合并分支
git checkout -b 创建并切换分支
git branch -a 查看所有分支
git branch -v 产看本地分支

远程仓库

git remote add origin xx 添加远程仓库
