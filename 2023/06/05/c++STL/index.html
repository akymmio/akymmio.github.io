c++STL

发表于2023-06-05|更新于2023-12-11|c++
|阅读量:

记录一些常用的STL方法,方便忘记的时候查找记忆

一.vector

1.创建和初始化对象

  vector<int> v1;                        // 执行默认构造函数，为空
  vector<int> v2(v1);                    // 拷贝构造（容器类型和元素类型必须相同）
  vector<int> v3 = v1;                   // 为空
  vector<int> v4{1, 2, 3};               // 初始化列表，3个元素: 1, 2, 3
  vector<int> v5 = {1, 2, 3};            // 3个元素: 1, 2, 3
  vector<int> v6(v5.begin(), v5.end());  // 传递迭代器参数进行拷贝构造，1，2，3
                                         // 容器类型和元素可以不相同

  vector<int> v7(5);                     // 仅适用于顺序容器，5个元素: 0, 0, 0, 0, 0
  vector<int> v8(5, 1);                  // 仅适用于顺序容器，5个元素: 1, 1, 1, 1, 1

2.基本操作

函数 备注
capacity()
size()
push_back()
pop_back()
front()
back()
begin()
end()
insert(pos,elem) pos为迭代器
insert(pos,n,elem)
erase(v.begin(),v.end())
clear()
vector<\int>::iterator iter=v.begin();获得迭代器

map/multimap

1.介绍

  • map为单重映射，multiple为多重映射（一个键对应多个值）
  • map：红黑树实现，自动排序
  • unordered_map：哈希表实现，无序

2.创建对象

  map<int,int> m;
  map<int,int,op> //op为排序规则

3.基本操作

函数 含义
m[key]=value 赋值
erase(it) 删除迭代器对应的键和值
erase(key) 删除键和对应的值
erase(first,lase) 删除左闭右开的区间对应的键值
clear() 清除所有元素
count(key) 查看元素是否存在，返回1，0
find(key) 返回key映射的迭代器
lower_bound() 返回一个迭代器，指向键值>= key的第一个元素
upper_bound() 返回一个迭代器，指向键值> key的第一个元素

4.添加元素

  1.map[key]=value;

  2.mp.insert(pair<int,int>(value,value));

  3.mp.insert({value,value});

  4.mp.insert(make_pair(value,value));

queue

priority_queue

只是在队列的基础上有了排序
可以以O(log n) 的效率查找一个队列中的最大值或者最小值（堆结构）

  //升序队列
  priority_queue <int,vector<int>,greater<int> > q;
  //降序队列（默认大顶堆）
  priority_queue <int,vector<int>,less<int> >q;
