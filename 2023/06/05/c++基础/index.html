位运算

符号 描述 运算规则
& 与 两个为1，结果为1（有0则0）
| 或 两个为0，结果为0（有1则1）
^ 异或 不同为1，相同为0

1& 取对应位置上的数
0| 加上对应位置的数
lowbit=tmp&-tmp; 取tmp的最低位1

const

1.const含义

  • 常类型是指使用类型修饰符const说明的类型，常类型的变量或对象的值是不能被更新的。

2.const对象默认为文件局部变量

  • const修饰的变量要显示地声明为const，做初始化

3.定义常量

4.const指针

  • int *const p，指针常量 指针本身是常量，指向固定地址，内容可以变 必须初始化

  • const int *p，常量指针 本身是指针变量，指向的内容不可以变，指向可以变

    如果const位于*的左侧，const就是用来修饰指针所指向的变量，即指针指向为常量；
    如果const位于*的右侧，const就是修饰指针本身，即指针本身是常量。

5.类中使用const

  • 不修改数据成员的函数应该声明为const

  • const对象只能访问const成员函数,而非const对象可以访问任意的成员函数,包括const成员函数.

      int Apple::getCount() const
  {
      take(1);
      add(); //error(add为普通函数)
      return apple_number;
  }

  • 对于类中的const成员变量必须通过初始化列表进行初始化，

      class Apple
  {
      private:
          int people[100];
      public:
          Apple(int i); 
          const int apple_number;
  };
  Apple::Apple(int i):apple_number(i)
  {
  }
